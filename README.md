### Setting Up Code Consistency Tools

-Duster by Titan. Duster bundles several linters and fixers together: - Laravel's Pint - TLint (handles Laravel's Blade) - PHP Code Sniffer - PHP CS Fixer

- GitHub Actions run linters/fixers on CI when someone pushes a commit.
- Husky hooks run linters/fixers locally before you commit.
- Prettier
- Mailpit (Docker)
